# Cohab-backend

## Семейный список дел
Ссылка на веб сайт: https://magofrays.sknt.ru/cohab/ 
## Текущий функционал:
- Регистрация пользователей
- Создание и редактирование семьи
- Приглашение других пользователей в семью
- Создание задач и назначение их на пользователей, состоящих в семье
- Отметка задачи выполненной и отметка задачи проверенной
- Отправка уведомлений пользователям при модификации семьи/задач
- Создание ролей в семье с соответстсвующими доступами
- Назначение на пользователей ролей с соответствующими доступами

Используемый стек: Java, Spring, PostgreSQL, Kafka
